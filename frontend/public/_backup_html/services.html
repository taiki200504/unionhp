学生・学生団体向けサービス

UNIONは学生の声を届け、活動を支援するためのサービスを提供しています。
加盟団体は以下のサポートを受けることができます。

Podcast出演

学生団体や個人の活動・思いを音声メディアで発信。Apple Podcast / Spotify / YouTubeで配信されるユニラジに出演できます。

#Podcast #加盟団体無料
出演申込

学生団体ニュース

学生の声を社会へ届ける。UNIONでは、加盟団体や全国の学生団体の活動を縦型動画コンテンツとして発信するニュース企画を立ち上げました。毎週のショートニュースとして、若者のリアルな挑戦やアクションを社会とつなぎます。

#ニュース #動画 #学生団体
プロジェクト詳細

告知サポート

Instagram / Note / X（旧Twitter）などでイベント・取り組みの広報。フォームから依頼可能です。加盟団体の活動をUNIONのSNSでシェア。

#SNS広報 #イベント告知
告知依頼

学生オンリーSlack

情報交換・募集告知・交流のハブ。月1回の団体代表ミーティングやイベント情報の共有、他団体との連携を促進します。

#コミュニティ #情報交換
Slack参加

1on1カウンセリング

リーダー専用サポート窓口。組織運営や悩みの個別相談。人事・法務・広報のプロフェッショナルが対応します。

#リーダー支援 #組織運営
相談予約

コラボレーション

他団体や企業との協業機会提供。「UNION Lab」等のプロジェクト共有スペース活用。イベント連携も促進します。

#マッチング #共創
問い合わせ

インターン紹介

協賛企業とのインターン紹介。Slack・個別DMにて学生に紹介。メディア運営や制作に関わるOJTのチャンスも提供。
